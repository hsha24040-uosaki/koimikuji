恋御籤（五条悟）大丈夫僕最強だからアプリ
このプロジェクトは、ボタンをクリックすることで今日の恋愛運を占うことができるシンプルなWebアプリケーションです。某人気キャラクターの雰囲気を楽しみながら、ランダムに運勢が表示されます。

構成ファイル
プロジェクトは以下の3つのファイルで構成されています。

index.html: アプリケーションの構造（HTML）

style.css: アプリケーションのデザイン（CSS）

script.js: おみくじの抽選ロジック（JavaScript）

機能
「おみくじを引く」ボタンをクリックすると、JavaScriptで定義された5種類の運勢（大吉、吉、中吉、小吉、凶）からランダムに1つが画面に表示されます。

レスポンシブで清潔感のあるパステルカラーのデザインを採用しています。

使い方
全てのファイル（index.html, style.css, script.js）を同じフォルダに保存します。

ブラウザで index.html を開きます。

「おみくじを引く」ボタンを押して運勢を確認してください。

技術スタック
HTML5: 構造の定義

CSS3: スタイリング（ボタンのホバーエフェクト、背景色の設定など）

JavaScript (ES6): イベントリスナーを用いたランダム抽選処理
